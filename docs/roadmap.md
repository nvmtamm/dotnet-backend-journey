# BookVerse Roadmap

This roadmap breaks the learning journey into small, commit-friendly milestones.

## Phase 1: C# Fundamentals

| Day | Topic | Practice |
|---|---|---|
| Day 01 | Variables and Data Types | Store book information with basic C# types |
| Day 02 | Input and Output | Read book data from the console |
| Day 03 | Operators and Expressions | Calculate discounts, totals, and stock value |
| Day 04 | Conditions | Check book availability and pricing rules |
| Day 05 | Switch Statements | Build simple menu choices |
| Day 06 | Loops | Print book lists and repeat menu actions |
| Day 07 | Methods | Split logic into reusable methods |

## Phase 2: Collections and LINQ

| Topic | Practice |
|---|---|
| Array and List | Store multiple books |
| Dictionary | Store books by ID |
| Lambda | Write short filter conditions |
| LINQ | Search, filter, sort, and project book data |

## Phase 3: Object-Oriented Programming

| Topic | Practice |
|---|---|
| Class and Object | Create Book, Category, Customer models |
| Constructor and Property | Initialize objects cleanly |
| Encapsulation | Protect internal state |
| Inheritance | Create shared base models |
| Interface | Define repository contracts |
| Abstract Class | Share common behavior |
| Polymorphism | Handle different payment/order types |

## Phase 4: BookVerse Console App

- Add book
- Update book
- Delete book
- Search book
- Show all books
- Manage cart
- Create order

## Phase 5: ASP.NET Core Backend

- Entity Framework Core
- SQL Server
- ASP.NET Core Web API
- JWT Authentication
- Role-based Authorization
- Clean Architecture
- Unit Testing
